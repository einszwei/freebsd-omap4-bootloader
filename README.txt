freebsd-omap4-bootloader

Usage

1. Dowonload the freebsd-omap4-bootloader on sys/boot/arm/omap4
2. Copy the build.sh to the upper directory of the freebsd source directory.
3. Execute ./build boot
4. The MLO is created on out/obj/arm.armv6/${WORKING_DIRTORY}/sys/boot/arm/omap4/boot1/
5. About gpt partion, reffer to http://yagstore.blogspot.kr/2015/07/freebsd-gpt-partiton-on-pandaboard.html

For now, the serial log is available and Usage will be changed.
